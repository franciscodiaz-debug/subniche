# SubNiche

SubNiche is a niche-first marketplace for enthusiast communities to buy, sell, trade, showcase collections, and connect.

This repository is the canonical source of truth for the product. v0, Claude Design, Figma, screenshots, and other prototypes are reference material only unless explicitly promoted into this repo.

## Core product idea

SubNiche is not a generic marketplace with categories. It is a platform for focused enthusiast markets where identity, taste, knowledge, collections, trade interests, and community context matter.

The product combines:

- niche-specific marketplaces
- user collections and wishlists
- trading as a first-class behavior
- community-specific market surfaces
- user profiles with trust context
- eventual discussion/community features

## Development philosophy

Build from shared product rules, design-system rules, and reusable components.

Do not build isolated one-off pages that drift from the rest of the product.
