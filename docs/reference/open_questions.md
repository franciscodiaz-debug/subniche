# Prototype Open Questions

## Access and Evidence

- Deployment protection was disabled for the current observed URLs. Should future prototype observation continue from public preview deployments, or should we prefer local v0 exports when available?

## Trade Interest

- User stated trade-interest setting functionality is incomplete. What is the intended MVP flow for setting trade interest?
- Is trade interest attached to an item, a listing, a user want-list, or a combination?
- Are trade criteria category-only, attribute-level, free-text, or saved-search based?

## Trade Matches

- What score thresholds define perfect, strong, weak, and inbound-only cards?
- Should inbound interest appear in the same trade grid or a separate section/tab?
- Can inbound interest become a true match after the owner adds matching trade interest?

## Offers

- User stated the inbox offer flow is incomplete. What are the final MVP rules for accept, counter, decline, and messaging?
- Does accepting or declining require a confirmation step?
- Can a counter offer change item/cash terms, or only add a note?
- Are offers always tied to an inbox thread?

## Listing Detail

- What should the share control do: native share, copied-link toast, custom popover, or all depending on browser/device?
- Should the message action route to inbox directly or open an inline composer?
- Is the unrelated gallery image in the prototype accepted placeholder drift or a bug?

## My Stuff

- Which item actions are required for MVP: edit, duplicate, archive, delete, mark sold, change visibility, set trade interest?
- How should private and uncategorized item states affect public profile visibility?

## Add Item

- Does `Add Item` publish immediately, route to a review screen, or save and redirect to My Stuff?
- What validation is required before saving a draft versus adding/publishing an item?
- Should `AI Assist` be enabled in MVP, and what source data should it use?

## Home and Onboarding

- Does `I'll do this later` permanently dismiss onboarding or just defer it?
- Which home modules are MVP-critical?
- Should preview media be video, static thumbnail, or omitted?
