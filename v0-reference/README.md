# V0 Reference Archive

This folder stores V0 screenshots and exported prototype code as reference material only.

Do not import this code directly into the production app. Use it later during the design-system audit and page implementation branches to identify useful patterns, conflicts, and reusable ideas.

## Structure

- `homepage-logged-out/` - logged-out marketing/home surface reference
- `homepage-logged-in/` - authenticated home/feed reference
- `marketplace/` - browse, grid, filters, and market mode reference
- `profile/` - user profile, activity, collections, wants, and trust surface reference
- `add-item/` - create listing/add item flow reference
- `listing-detail/` - listing detail reference placeholder
- `collections/` - my stuff and collection detail reference screenshots
- `communities/` - community reference placeholder
- `inbox/` - inbox/messages reference placeholder
- `trade/` - trade mode, trade menu, and trade offer planning reference
- `full-project/` - full V0 project export for broader page/component reference
- `loose-files/` - related V0/downloaded files that do not map cleanly to one page
